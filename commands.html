ONTAP CLI

Commands Reference

ONTAP CLI commands organized by category. Use the filter to find commands by keyword or type.

Command Description Example / Notes Category
volume show Display volumes with state, size, and usage volume show -fields name,size,state,space-used Volume
volume create Create a new FlexVol in an aggregate and SVM volume create -vserver svm0 -volume vol1 -aggregate aggr1 -size 10g -junction-path /vol1 Volume
volume modify Change volume properties such as size, autosize, or tiering volume modify -vserver svm0 -volume vol1 -size 20g Volume
volume delete Delete a volume (must be offline first) volume offline -vserver svm0 -volume vol1; volume delete -vserver svm0 -volume vol1 Volume
volume move start Move a volume to another aggregate non-disruptively volume move start -vserver svm0 -volume vol1 -destination-aggregate aggr2 Volume
volume efficiency on Enable deduplication and compression on a volume volume efficiency on -vserver svm0 -volume vol1 Volume
volume quota show Display disk and file quota limits volume quota show -vserver svm0 -volume vol1 Volume
volume snapshot create Create a point-in-time snapshot of a volume volume snapshot create -vserver svm0 -volume vol1 -snapshot snap1 Snapshot
volume snapshot show List snapshots for a volume with size and creation time volume snapshot show -vserver svm0 -volume vol1 Snapshot
volume snapshot delete Delete one or more snapshots volume snapshot delete -vserver svm0 -volume vol1 -snapshot snap1 Snapshot
volume snapshot restore Restore a volume to a snapshot state (disruptive) volume snapshot restore -vserver svm0 -volume vol1 -snapshot snap1 Snapshot
volume snapshot restore-file Restore a single file from a snapshot without full restore volume snapshot restore-file -vserver svm0 -volume vol1 -snapshot snap1 -path /dir/file.txt Snapshot
snapmirror show Display SnapMirror relationships, state, and lag snapmirror show -fields source-path,destination-path,lag-time Snapshot
snapmirror update Trigger a manual SnapMirror update (on-demand sync) snapmirror update -source-path svm0:vol1 -destination-path svm1:vol1_dr Snapshot
snapmirror initialize Initialize a new SnapMirror relationship (baseline transfer) snapmirror initialize -destination-path svm1:vol1_dr Snapshot
network interface show Show all LIFs, their IP, port, and status network interface show -vserver svm0 Network
network interface create Create a new logical interface (LIF) network interface create -vserver svm0 -lif lif1 -address 10.0.0.10 -netmask 255.255.255.0 -home-node node1 -home-port e0a Network
network interface migrate Migrate a LIF to another port non-disruptively network interface migrate -vserver svm0 -lif lif1 -dest-node node2 -dest-port e0b Network
network port show List physical and VLAN ports on each node network port show -fields port,link,speed,health-status Network
network route show Display routing table entries per SVM network route show -vserver svm0 Network
vserver show List all SVMs with type, state, and allowed protocols vserver show -fields name,state,allowed-protocols SVM
vserver create Create a new SVM (data vserver) vserver create -vserver svm_new -aggregate aggr1 -rootvolume svm_new_root -rootvolume-security-style unix SVM
vserver nfs show Show NFS configuration for an SVM vserver nfs show -vserver svm0 SVM
vserver export-policy show Display export policies and their rules vserver export-policy show -vserver svm0 SVM
system node show Show node health, OS version, uptime, and model system node show -fields node,health,model,uptime Diagnostics
cluster show Show cluster nodes and overall health status cluster show Diagnostics
storage aggregate show Show aggregate capacity, state, and disk info storage aggregate show -fields availsize,usedsize,state Diagnostics
event log show Display cluster event messages by severity or time event log show -severity error -time-range 1h Diagnostics
security audit log show View admin activity for security compliance auditing security audit log show -user admin Diagnostics
qos policy-group show View QoS policy groups and their IOPS/throughput limits qos policy-group show Diagnostics
statistics show-periodic Live performance counter polling for a specific object statistics show-periodic -object volume -instance vol1 -counter read_ops,write_ops -interval 5 Diagnostics
